
# Connectify

Connectify is a web application designed to connect people through seamless communication and collaboration. The platform allows users to create profiles, post updates, and interact with others through real-time messaging, making it a one-stop solution for staying connected with your network.

## Live Demo

Check out the live version of the project [here](https://connectify-xi-plum.vercel.app/).

## Features

- **User Authentication**: Secure sign-up, login, and logout functionality.
- **Profile Management**: Create, edit, and view user profiles.
- **Real-Time Messaging**: Send and receive instant messages with other users.
- **Posts and Feeds**: Share posts, media, and updates on the main feed.
- **Notifications**: Receive real-time notifications for messages and other activities.
- **Responsive Design**: Fully responsive layout for a seamless experience across devices.

## Technologies Used

- **Frontend**: ReactJS
- **Backend**: Node.js, Express.js
- **Database**: MongoDB
- **Real-Time Communication**: Socket.IO for real-time messaging
- **Hosting**: Vercel
- **Styling**: Tailwind CSS

## Installation

To run this project locally, follow these steps:

1. **Clone the repository:**
   ```bash
   git clone https://github.com/your-username/connectify.git

2. **Navigate to the project directory:**
   ```bash
   cd connectify
   
3. **Install dependencies both frontend and backend:**
    ```bash
   npm install
    
4. **Set up environment variables in backend:**
   ```bash
   MONGODB_URI=your_mongodb_connection_string
   JWT_SECRET=your_jwt_secret

5. **Run the development server:**
   ```bash
    npm start
